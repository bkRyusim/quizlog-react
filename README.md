# quizlog-go

Frontend code of `Quizlog`.

Project for graduation in `Design Projects(Capstone Design)` in `CAU`.

## About Quizlog

`Quizlog` is Quiz platform based on user's posts on `Tistory`. Users can make their own Quiz for self-study.

## Project Stack

### Frontend

- `javascript`
- `react.js`

### Backend

- `golang`
- `fiber`
- `ent`
- `facebookgo/inject`

## Try code

npm start
